# Demo Workflow

1. Launch the dashboard with `streamlit run app.py`.
2. Open the `Raw Measurement` tab and import `examples/sample_touchstone.s1p`.
3. Review the magnitude, phase, and Smith chart traces.
4. Upload open, short, and load standards in the `Calibration` tab if you have them.
5. Pick a sensor geometry in `Sensor Geometry`.
6. Inspect `Material Properties` to review `epsilon'`, `epsilon''`, loss tangent, conductivity, Nyquist, impedance, and admittance plots.
7. Export the dielectric spectrum as CSV or export any Plotly chart as PNG/SVG.
