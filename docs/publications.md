# Publications and Citation

LitePerm is designed for academic and industrial research workflows. If you use it in a thesis, paper, preprint, or technical report, include a software citation.

## Preferred Citation

LitePerm ships with a root-level `CITATION.cff` file for GitHub and reference-manager support.

## BibTeX Example

```bibtex
@software{croft_liteperm_2026,
  author  = {Croft, Dion},
  title   = {LitePerm},
  year    = {2026},
  version = {0.4.0},
  url     = {https://github.com/DionCroft/LitePerm}
}
```

## Academic Usage Guidance

When publishing LitePerm-based work, consider reporting:

- LitePerm version
- calibration method
- sensor geometry
- frequency range
- transform plugin or inverse solver used
- whether data was measured, synthetic, or browser-demo derived
