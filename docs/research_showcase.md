## Suggested Figure Set

For each showcase study, capture:

- raw S11 magnitude and phase
- Smith chart
- dielectric spectrum
- inverse residual and convergence plots where applicable
- experiment metadata summary

## Recommended Publication Path

1. Save the experiment in `Research Mode`
2. Export figures as SVG or PDF
3. Add experiment metadata and sample file links
4. Cite LitePerm using the [Publications](publications.md) page and `CITATION.cff`
