# Roadmap

## Completed

### Platform Foundation

- GitHub Pages documentation portal
- browser-based LitePerm Web demo
- public downloads, release notes, and citation metadata
- deployment-ready documentation workflow for open-source collaboration

### Phase 2

- Direct LiteVNA integration
- Research-mode experiment storage
- Plugin-based transformation engine
- Sensor model framework
- Material database
- Automated reporting
- AI-prep interfaces
- FastAPI layer

### Phase 3

- Inverse electromagnetic modelling engine
- Forward models for patch, OECP, microstrip, and generic resonator studies
- Uncertainty quantification, sensitivity analysis, and parameter sweeps
- Synthetic dataset generation and digital twin updates

### Phase 4

- full-wave solver integration layer
- `SimulationJob` and `SimulationResult` models
- solver registry with status reporting
- openEMS adapter scaffold
- Meep adapter scaffold
- simulation caching under project archives
- measured-versus-simulated S11 comparison workflows

## Planned

### Phase 5

- real-time LiteVNA streaming with continuous inverse updates
- deeper openEMS automation
- Meep execution support beyond the current scaffold
- digital twin synchronisation improvements

### Phase 6

- AI-assisted material classification
- neural inverse models
- surrogate-model training workflows
- physics-informed hybrid experiments

### Phase 7

- patch antenna inverse modelling refinements
- solver-specific mesh and boundary presets
- HFSS, CST, and COMSOL enterprise adapter work

### Phase 8

- metamaterial sensor support
- CSRR and advanced resonator libraries

### Phase 9

- biomedical implant sensor support
- passive wireless implant sensing workflows
- tissue-model inverse estimation refinement

### Phase 10

- cloud-based experiment repository
- remote simulation clusters
- scalable research collaboration tooling
