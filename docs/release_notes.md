# Release Notes

## v0.5.0

LitePerm Phase 4 adds the first full-wave solver integration layer.

Highlights:

- `liteperm.solvers` package with job, result, registry, cache, export, and validation helpers
- `FullWaveForwardModel` for analytical, cached, and solver-backed workflows
- `Full-Wave Simulation` dashboard tab with solver status, setup, export, and comparison views
- openEMS adapter scaffold and Meep adapter scaffold
- simulation workflow docs and example job files

## v0.4.0

LitePerm Phase 4 turns the repository into a public-facing open-source project portal.

Highlights:

- MkDocs Material documentation site
- GitHub Pages deployment workflow
- browser-only Plotly.js demo
- installation guides for Windows 11, Ubuntu, and WSL2
- downloads page, citation metadata, and research showcase structure

## v0.3.0

- inverse electromagnetic modelling engine
- forward-model registry
- inverse solvers
- uncertainty, sensitivity, and parameter sweep tooling
- digital twin persistence

## v0.2.0

- direct LiteVNA integration architecture
- live sweep capture
- experiment database and project archive structure
- plugin-based transformation framework
- FastAPI service layer

## v0.1.0

- base Streamlit dashboard
- Touchstone and CSV import
- S11, impedance, admittance, and dielectric-spectrum views
- calibration and geometry profile support
